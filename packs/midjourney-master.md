## Parameter Cheat Sheet

### Aspect Ratios (--ar)
| Ratio | Use Case |
|-------|----------|
| `--ar 1:1` | Instagram posts, profile pics, logos |
| `--ar 4:5` | Instagram portrait, product shots |
| `--ar 2:3` | Portrait photography, posters |
| `--ar 3:2` | Landscape photography, blog headers |
| `--ar 16:9` | Desktop wallpapers, YouTube thumbnails |
| `--ar 9:16` | Stories, TikTok, mobile wallpapers |
| `--ar 21:9` | Ultrawide, cinematic banners |
| `--ar 3:1` | Twitter/X headers, website banners |

### Quality & Style
| Parameter | Effect |
|-----------|--------|
| `--v 6` | Latest Midjourney model |
| `--niji 6` | Anime/illustration optimized |
| `--style raw` | Less beautified, more photographic |
| `--q 2` | Higher quality (2x compute) |
| `--s 0-1000` | Stylize amount (0=literal, 1000=artistic) |
| `--c 0-100` | Chaos/variety (higher=more unexpected) |

### Camera & Lens Simulation
| Keyword | Effect |
|---------|--------|
| `shot on Hasselblad` | Medium format look, rich detail |
| `Canon R5 85mm f/1.4` | Creamy bokeh, sharp subject |
| `Leica Q3` | Street photography character |
| `35mm film` | Analog grain and color |
| `drone photography` | Aerial perspective |
| `tilt-shift` | Miniature effect |
| `macro lens` | Extreme close-up detail |
| `fisheye lens` | Ultra-wide distortion |

### Lighting Keywords
| Keyword | Effect |
|---------|--------|
| `golden hour` | Warm, directional sun |
| `blue hour` | Cool twilight tones |
| `Rembrandt lighting` | Dramatic triangle on cheek |
| `rim lighting` | Glowing edges |
| `volumetric light` | God rays, atmospheric |
| `neon lighting` | Cyberpunk/club vibes |
| `soft diffused light` | Even, gentle illumination |
| `harsh direct light` | Strong shadows, high contrast |
| `candlelight` | Warm, intimate, flickering |
| `studio lighting` | Clean, professional |

## Style Reference Guide

### Photography Styles
- **Editorial**: Fashion magazine quality, dramatic, styled
- **Documentary**: Raw, candid, storytelling
- **Fine Art**: Gallery-worthy, conceptual, intentional
- **Commercial**: Clean, bright, selling a product
- **Street**: Urban, candid, moment-captured

### Art Styles
- **Impressionist**: Monet, soft brushstrokes, light play
- **Art Nouveau**: Mucha, organic curves, decorative
- **Bauhaus**: Geometric, primary colors, functional
- **Vaporwave**: Neon, 80s aesthetic, digital nostalgia
- **Brutalist**: Raw concrete, bold geometry, powerful

### Color Palettes
- **Moody**: Deep blues, rich blacks, amber accents
- **Pastel**: Soft pink, mint, lavender, cream
- **Earth**: Terracotta, sage, sand, warm brown
- **Neon**: Hot pink, electric blue, toxic green
- **Monochrome**: Single hue variations, sophisticated
