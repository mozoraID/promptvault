## Landing Page Copy

```
Write a high-converting landing page for [product/service]. Target audience: [audience]. Price: [price]. Unique selling point: [USP]. Include: attention-grabbing headline (5 options), sub-headline, hero section copy, 3 benefit blocks with icons, social proof section, FAQ (5 questions), and final CTA. Tone: [confident/friendly/urgent]. Optimize for [conversion goal].
```

```
Write above-the-fold copy for a SaaS landing page. Product: [product]. Solves: [problem]. For: [audience]. Include: headline with specific outcome, sub-headline addressing #1 objection, 3-word benefit badges, CTA button text (3 options), and micro-copy below button to reduce friction.
```

```
Create a "Features vs Benefits" section for my landing page. Product: [product]. Features: [list features]. For each feature, write: benefit headline (outcome-focused), 2-sentence explanation, and an icon suggestion. Format for a 3-column grid layout.
```

```
Write a testimonial request email and template. My product: [product]. Customer results: [typical results]. Give me: the ask email, a fill-in-the-blank template customers can use, and 5 example testimonials I can use as social proof (clearly marked as examples to rewrite with real customer data).
```

```
Create a pricing section for my landing page. Plans: [list plans with features]. Write: plan names (creative, not just Basic/Pro/Enterprise), feature lists with checkmarks, recommended plan badge copy, CTA for each tier, and FAQ about pricing (billing, refunds, upgrades).
```

```
Write a "How It Works" section. Process: [describe your process in 3-5 steps]. For each step: catchy step title, one-sentence description, and supporting detail. Add an intro sentence and closing CTA. Keep it scannable.
```

```
Create an exit-intent popup copy for [website]. Offer: [lead magnet/discount]. Write: attention-grabbing headline, 2-line description, email input placeholder text, CTA button text, and "no thanks" link text that creates subtle FOMO.
```

```
Write comparison landing page copy: "[Your Product] vs [Competitor]". Structure: intro acknowledging both options, comparison table (10 features), "Why customers switch" section (3 stories), and honest "When [competitor] might be better" section (builds trust). CTA: free trial.
```

## Email Sequences

### Welcome Sequence (5 emails)
```
Write a 5-email welcome sequence for new subscribers who downloaded [lead magnet]. Business: [business]. Product to eventually sell: [product] at [price]. 

Email 1 (Day 0): Deliver the lead magnet + quick win tip. Subject line should get 50%+ open rate.
Email 2 (Day 2): Your origin story — why you started [business]. Build connection and credibility.
Email 3 (Day 4): The #1 mistake [audience] makes with [topic]. Educational + positions your product as solution.
Email 4 (Day 7): Case study/transformation story. Before → After with specific numbers.
Email 5 (Day 10): Soft pitch for [product]. Frame as "next step" not hard sell. Include limited-time bonus.

For each email include: subject line (+ A/B variant), preview text, full body copy, CTA, and P.S. line.
```

### Sales Sequence (7 emails)
```
Write a 7-email sales sequence for launching [product] at [price]. Audience: [audience] who are already on my email list. Use the PASTOR framework across the sequence.

Email 1 (Day 0): Problem identification — paint the pain vividly
Email 2 (Day 1): Amplify — what happens if they don't solve this
Email 3 (Day 2): Story — transformation story of someone who solved it  
Email 4 (Day 3): Transformation — what life looks like on the other side
Email 5 (Day 4): Offer reveal — full details, bonuses, pricing
Email 6 (Day 6): Objection crusher — address top 5 objections
Email 7 (Day 7): Deadline — last chance, scarcity, urgency

Include subject lines, body copy, and CTAs for each. Tone: [direct/friendly/authoritative].
```

### Re-engagement Sequence
```
Write a 3-email re-engagement sequence for subscribers who haven't opened in 60+ days. Business: [business].

Email 1: "Are you still there?" — curiosity + value reminder
Email 2: "I made this for you" — exclusive content/offer only for inactive subscribers
Email 3: "Cleaning up my list" — honest FOMO, last chance before removal

Include: subject lines designed to break through inbox blindness, and a clear CTA in each.
```

## Ad Copy

### Facebook/Instagram Ads
```
Write 5 Facebook ad variations for [product/service]. Target: [audience demographic + psychographic]. Goal: [conversions/leads/traffic]. Budget context: [small/medium/large].

For each ad: primary text (short + long version), headline, description, CTA button recommendation, and suggested image/video direction. Use these frameworks across the 5 ads: pain-agitate-solve, story, testimonial, comparison, and direct offer.
```

```
Create 3 Facebook ad funnels for [product]. For each funnel stage:
- Top of funnel: brand awareness ad (video concept + copy)
- Middle of funnel: retargeting ad for website visitors (social proof + copy)  
- Bottom of funnel: conversion ad for cart abandoners (urgency + copy)
Include audience targeting suggestions and budget split recommendation.
```

### Google Ads
```
Create a Google Ads campaign structure for [business]. Monthly budget: [budget].

Include:
- 5 ad groups with themes
- 10 keywords per ad group (with match types)
- 20 negative keywords
- 3 responsive search ads per ad group (15 headlines, 4 descriptions each)
- Sitelink extensions (4)
- Callout extensions (4)
- Bidding strategy recommendation
```

```
Write Google Ads copy for [product/service] targeting "[primary keyword]". Include: 15 headline options (30 chars each), 4 description options (90 chars each), display URL path suggestions, and recommended extensions. Focus on: specific outcomes, numbers, and urgency.
```

### LinkedIn Ads
```
Write 3 LinkedIn sponsored post ads for [B2B product/service]. Target: [job titles] at [company size] in [industry].

For each: hook question/statement, body copy (under 150 words), CTA, and suggested content format (single image, carousel, or video). One should be thought-leadership style, one direct response, and one case study.
```

## Financial Models

```
Create a 12-month financial projection for my [business type]. Starting point: [current revenue/stage]. Assumptions: [growth rate, pricing, costs]. Build a month-by-month table with: revenue (broken by source), COGS, gross profit, operating expenses (categorized: marketing, tools, contractors, other), net profit, and cash balance. Include break-even analysis.
```

```
Help me calculate unit economics for [product/service]. Price: [price]. Current metrics: [what you know]. Calculate and explain: Customer Acquisition Cost (CAC), Lifetime Value (LTV), LTV:CAC ratio, payback period, gross margin, and contribution margin. Tell me if my economics are healthy and what to improve.
```

```
Create a pricing strategy analysis for [product]. Current price: [price]. Competitors: [competitor prices]. Value delivered: [describe value]. Analyze: price elasticity estimation, value-based pricing calculation, psychological pricing options, bundle pricing strategy, and recommended price with justification.
```

```
Build a freelancer/consultant rate calculator. My skills: [skills]. Experience: [years]. Market: [location/remote]. Calculate: hourly rate, daily rate, project-based pricing framework, retainer pricing, and value-based pricing for different project types. Include how to handle scope creep and rate negotiations.
```

```
Create a SaaS metrics dashboard template. Include formulas and explanations for: MRR, ARR, churn rate, expansion revenue, net revenue retention, CAC, LTV, CAC payback period, Rule of 40 score, and burn multiple. Show me what "good" looks like for each at my stage: [stage/ARR level].
```
